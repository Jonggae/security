# Spring 시큐리티

* Spring Security를 이용하여 회원가입 / 로그인 기능을 구현해본다.

---

## 프로젝트 구성

**Version** : SpringBoot 3.2.1

**Database** : H2 database

**Dependencies** : lombok, web, security, thymeleaf, validation etc.

---

## 구현 기능

**1. 회원가입**

* usernmae, password, email 을 가지는 User entity를 생성하여 DB에 저장

**2. 로그인**

* JWT 인증을 활용한 로그인
